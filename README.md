# 💰 Financial Chatbot Prototype

An AI-powered financial chatbot developed in Python that analyzes company financial data and provides insights.  
This project demonstrates basic financial analysis using data processing and visualization.

---

## 📌 Project Overview

The Financial Chatbot reads financial data from a CSV dataset and allows users to analyze company performance such as revenue, net income, assets, and liabilities.

This project was developed as part of a **Generative AI job simulation** to demonstrate data analysis and chatbot logic.

---

## 🚀 Features

- Read financial data from CSV files
- Analyze company financial performance
- Generate financial insights
- Simple command-line chatbot interface
- Data visualization using graphs

---

## 🛠 Technologies Used

- Python
- Pandas
- Matplotlib
